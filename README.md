# 114-1 期末專案：顧客資料分析

## 專案簡介
114-1 智慧商務系必修課程：大數據商業分析與決策

本專案旨在參與期末資料分析競賽，透過分析顧客資料，深入探討顧客流失、合約類型、顧客分群及市場滲透率等議題，並從中發現有價值的洞察。
分析方法涵蓋探索性資料分析、機器學習模型、分群分析、關聯規則探勘等技術。

## 檔案與目錄說明

### 資料檔案
- `customer_data.csv`: 原始顧客資料集。
- `customer_zip.csv`: 包含顧客郵遞區號資訊的資料。
- `cleaned_customer_data.csv`: 清洗及預處理後的顧客資料，可直接用於分析。

### Python 腳本
- `data_cleaning.py`: 清洗 `customer_data.csv` 和 `customer_zip.csv` 中的原始資料，產生 `cleaned_customer_data.csv`。
- `data_describe.py`: 生成資料的描述性統計摘要。
- `01.py`: 對所有顧客資料執行初步的探索性資料分析。
- `02.py`: 專注於分析不同優惠方式的有效性及特徵。

### 分析資料夾
- `03/`: **顧客狀態與流失分析**：包含分析顧客狀態並找出流失主要原因的腳本及結果。
- `04/`: **顧客合約類型決策樹**：使用機器學習（決策樹模型）對顧客合約類型進行分類和理解。
- `05/`: **顧客分群分析**：將顧客分組並分析各群組之間的特徵差異。
- `06/`: **年齡群組關聯規則分析**：識別不同年齡群組內的關聯規則及模式。
- `07_zip/`: **市場滲透率分析**：基於地理資料（郵遞區號）分析市場滲透率。
- `08_recommend/`: **推薦分析**：調查與推薦次數相關的關聯性。
