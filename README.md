# 114-1 期末專案：顧客資料分析

## 專案簡介
114-1 智慧商務系必修課程：大數據商業分析與決策

本專案旨在參與期末資料分析競賽，透過分析顧客資料，深入探討顧客流失、合約類型、顧客分群及市場滲透率等議題，並從中發現有價值的洞察。
分析方法涵蓋探索性資料分析、機器學習模型、分群分析、關聯規則探勘等技術。

## 資料說明

### 資料檔案
| 檔案名稱 | 說明 | 資料筆數 | 編碼 |
|---------|------|---------|------|
| `customer_data.csv` | 原始顧客資料集 | 7,043 筆 | Big5 |
| `customer_zip.csv` | 郵遞區號人口資料 | 1,664 筆 | Big5 |
| `cleaned_customer_data.csv` | 清洗後的顧客資料 | 7,043 筆 | UTF-8-sig |

### 資料欄位（38 個）
**基本資訊**：客戶編號、性別、年齡、婚姻、扶養人數、城市、郵遞區號、緯度、經度

**服務使用**：電話服務、多線路服務、網路服務、網路連線類型、線上安全/備份/設備保護/技術支援、電視/電影/音樂節目、無限資料下載

**財務資訊**：加入期間、合約類型、無紙化計費、支付方式、每月費用、總費用、總收入、額外費用、退款

**流失相關**：客戶狀態、客戶流失類別、客戶離開原因

## 主要成果

1. **流失預防**
   - 識別高風險客戶群（月租、高費用）
   - 主要流失原因：競爭對手、價格、服務不滿

2. **市場策略**
   - 東部市場：彈性方案、優惠導向
   - 西部市場：長約方案、家庭導向
   - 潛力市場：高人口低滲透率地區

3. **服務推薦**
   - 青年/中年：影音娛樂組合
   - 老年：電視電影組合
   - 基於客戶特徵的個性化推薦

4. **客戶價值**
   - CLV 分群精準識別高價值客戶
   - 優先留存 High CLV 客戶
   - 針對不同價值群組差異化策略
