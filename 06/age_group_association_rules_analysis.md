# 年齡群組關聯規則分析報告

## 一、分析概述

本報告針對公司顧客根據年齡進行分群（青年、中年、老年），並比較各年齡群組在使用公司服務時的關聯規則，分析其相同與相異之處。

### 1.1 年齡分群定義

- **青年群組（青）**：年齡 ≤ 40 歲
- **中年群組（中）**：40 歲 < 年齡 ≤ 65 歲  
- **老年群組（老）**：年齡 > 65 歲

### 1.2 分析方法

- **演算法**：Apriori 演算法
- **最小支持度**：0.1 (10%)
- **最小信賴度**：0.5 (50%)
- **最小提升度**：1.2

### 1.3 分析的服務項目

本分析涵蓋以下 12 項服務特徵：
1. 電話服務
2. 多線路服務
3. 網路服務
4. 線上安全服務
5. 線上備份服務
6. 設備保護計劃
7. 技術支援計劃
8. 電視節目
9. 電影節目
10. 音樂節目
11. 無限資料下載
12. 無紙化計費

## 二、各年齡群組基本統計

| 年齡群組 | 樣本數 | 規則數量 | 平均支持度 | 平均信賴度 | 平均提升度 | 最大提升度 |
|---------|--------|----------|-----------|-----------|-----------|-----------|
| 青 (≤40歲) | 2,817 | 9,952 | 0.140 | 0.702 | 1.978 | 3.519 |
| 中 (41-65歲) | 3,163 | 9,912 | 0.138 | 0.705 | 2.103 | 3.732 |
| 老 (>65歲) | 1,063 | 5,462 | 0.155 | 0.643 | 1.517 | 2.333 |

### 2.1 關鍵發現

1. **樣本數分布**：中年群組人數最多（3,163人），老年群組最少（1,063人）
2. **規則複雜度**：青年和中年群組產生的規則數量相近（約9,900條），老年群組規則數較少（5,462條）
3. **支持度**：老年群組的平均支持度最高（0.155），表示服務組合的普及率較高
4. **信賴度**：三個群組的信賴度都在 64-70% 之間，中年群組略高
5. **提升度**：中年群組的平均提升度和最大提升度均為最高，顯示該群組的服務關聯性最強

## 三、各年齡群組的典型關聯規則

### 3.1 青年群組（≤40歲）的典型規則

**最強關聯規則（提升度 3.52）：**
```
電影節目 + 設備保護計劃 + 電話服務 
  → 網路服務 + 音樂節目 + 電視節目 + 多線路服務
```
- 支持度：10.5%
- 信賴度：54.6%

**特徵分析：**
- 青年群組偏好**多媒體娛樂服務組合**（電影、音樂、電視）
- 對**設備保護計劃**有較高需求
- 傾向使用**多線路服務**，可能反映多設備使用習慣
- 服務組合複雜度高，喜歡「全套服務」

### 3.2 中年群組（41-65歲）的典型規則

**最強關聯規則（提升度 3.73）：**
```
音樂節目 + 電視節目 + 多線路服務 
  → 電影節目 + 設備保護計劃 + 電話服務
```
- 支持度：10.7%
- 信賴度：68.9%

**特徵分析：**
- 中年群組同樣重視**娛樂服務組合**
- **信賴度最高**（68.9%），顯示服務搭配模式最穩定
- 對**設備保護計劃**的需求明確
- **提升度最高**，表示服務間的關聯性最強

### 3.3 老年群組（>65歲）的典型規則

**最強關聯規則（提升度 2.33）：**
```
電影節目 + 設備保護計劃 + 多線路服務 
  → 線上備份服務 + 電話服務 + 電視節目
```
- 支持度：11.1%
- 信賴度：52.7%

**特徵分析：**
- 老年群組特別重視**線上備份服務**（在青年和中年群組較不明顯）
- **支持度最高**（11.1%），表示此模式在該群組中最普遍
- **提升度較低**（2.33），顯示服務選擇較為分散
- 對**資料安全和備份**有較高關注度

## 四、三個年齡群組的共同規則分析

### 4.1 共同規則特徵

分析發現，三個年齡群組存在**大量共同的服務使用模式**，主要集中在以下領域：

#### 1. 娛樂服務核心組合
所有年齡群組都顯示出以下共同模式：
- **電影 + 電視 + 音樂** 三項娛樂服務經常一起訂購
- 娛樂服務與**電話服務**高度關聯
- 娛樂服務與**網路服務**強關聯

#### 2. 基礎服務組合
- **電話服務 + 網路服務** 是所有群組的基礎配置
- **多線路服務**通常與網路服務搭配
- **無紙化計費**在所有群組中都普遍採用

#### 3. 增值服務模式
- **設備保護計劃**在所有年齡層都受歡迎
- **無限資料下載**與娛樂服務（電影、音樂）高度相關

### 4.2 共同規則示例

所有年齡群組都出現的典型規則：
```
1. 電影節目 → 網路服務
2. 音樂節目 → 電話服務
3. 電視節目 + 電影節目 → 網路服務
4. 無限資料下載 → 網路服務
5. 多線路服務 + 電話服務 → 網路服務
```

## 五、各年齡群組的獨有規則分析

### 5.1 青年群組獨有規則特徵

青年群組有**大量獨有規則**，主要特徵：

#### 核心差異點：
1. **技術支援計劃與電視節目的組合**
   - 青年群組顯示技術支援計劃經常與電視節目搭配
   - 可能反映年輕用戶對於新技術服務的嘗試意願

2. **複雜的服務組合**
   - 獨有規則多涉及 4-5 項服務的組合
   - 顯示青年用戶服務選擇更多元化

3. **無紙化計費的高關聯**
   - 無紙化計費在青年群組的規則中出現頻率更高
   - 反映數位化偏好

#### 代表性獨有規則：
```
技術支援計劃 + 網路服務 + 電視節目 → 多線路服務 + 電話服務
```

### 5.2 中年群組獨有規則特徵

中年群組的獨有規則顯示其**服務選擇最為穩定和均衡**：

#### 核心差異點：
1. **線上備份服務的特殊組合**
   - 中年群組顯示線上備份與多線路服務的特殊關聯
   - 可能反映對工作資料保護的需求

2. **娛樂與實用服務的平衡**
   - 獨有規則同時包含娛樂服務和保護類服務
   - 顯示中年用戶追求功能完整性

3. **多線路服務的核心地位**
   - 在獨有規則中，多線路服務經常作為前項條件
   - 可能反映家庭或工作多設備使用需求

#### 代表性獨有規則：
```
多線路服務 + 無限資料下載 + 線上備份服務 → 電影節目 + 網路服務
```

### 5.3 老年群組獨有規則特徵

老年群組的獨有規則最少，但有明確特徵：

#### 核心差異點：
1. **線上備份服務的核心地位**
   - **最顯著特徵**：線上備份服務在大量規則中出現
   - 與其他年齡層最大差異所在

2. **服務組合較簡單**
   - 獨有規則通常涉及 2-3 項服務
   - 偏好明確、簡單的服務組合

3. **對資料安全的重視**
   - 線上備份、設備保護頻繁出現
   - 反映對可靠性和安全性的重視

#### 代表性獨有規則：
```
設備保護計劃 + 電視節目 → 線上備份服務 + 電話服務
```

## 六、兩兩群組間的共同特徵

### 6.1 青年與中年共同（老年沒有）的規則

**共同點：117 條規則**

主要特徵：
1. **技術支援計劃的積極使用**
   - 兩個群組都顯示技術支援計劃的多樣化組合
   - 與電影、電視節目有強關聯

2. **複雜服務包的偏好**
   - 規則涉及 4-5 項服務的組合頻繁
   - 顯示活躍用戶的特徵

代表性規則：
```
技術支援計劃 + 網路服務 + 電影節目 → 多線路服務 + 電話服務
```

### 6.2 青年與老年共同（中年沒有）的規則

**共同點：117 條規則**

主要特徵：
1. **線上備份服務與設備保護的組合**
   - 顯示對資料安全的共同關注
   - 可能青年是因資料量大，老年是因安全意識

2. **無紙化計費與電視節目的特殊關聯**
   - 顯示數位化服務的採用模式

代表性規則：
```
無紙化計費 + 線上備份服務 + 設備保護計劃 → 網路服務 + 電視節目
```

### 6.3 中年與老年共同（青年沒有）的規則

**共同點：180 條規則**（最多）

主要特徵：
1. **多線路服務與娛樂服務的深度整合**
   - 大量規則涉及多線路 + 電影 + 電視的組合
   - 顯示家庭使用場景

2. **線上備份服務的特定使用模式**
   - 兩個群組共同顯示線上備份與娛樂服務的關聯
   - 可能反映對重要資料的長期保存需求

3. **無限資料下載的重要性**
   - 與娛樂服務強關聯
   - 顯示持續、大量的內容消費需求

代表性規則：
```
多線路服務 + 無限資料下載 + 電影節目 → 網路服務 + 電視節目 + 線上備份服務
```

## 七、綜合比較與洞察

### 7.1 服務偏好差異總結

| 特徵維度 | 青年群組 | 中年群組 | 老年群組 |
|---------|---------|---------|---------|
| **服務組合複雜度** | 高（4-5項） | 高（4-5項） | 中（2-3項） |
| **最重視的增值服務** | 設備保護 | 設備保護 + 線上備份 | 線上備份 |
| **技術支援計劃** | 積極使用 | 積極使用 | 較少使用 |
| **娛樂服務組合** | 全面（電影+電視+音樂） | 全面（電影+電視+音樂） | 選擇性（偏電視+電影） |
| **數位化程度** | 最高（無紙化） | 高 | 中 |
| **規則穩定性** | 中（信賴度70%） | 最高（信賴度71%） | 較低（信賴度64%） |

### 7.2 關鍵洞察

#### 1. 娛樂服務是核心驅動力
- **所有年齡層**都顯示娛樂服務（電影、電視、音樂）的強關聯
- 娛樂服務是吸引各年齡層客戶的共同點
- 建議：強化娛樂內容質量，作為客戶留存的核心策略

#### 2. 年齡影響服務複雜度
- **青年和中年**：偏好複雜、全面的服務組合
- **老年**：偏好簡單、明確的服務包
- 建議：針對不同年齡層設計不同複雜度的服務方案

#### 3. 資料安全重視度的差異
- **老年群組**：特別重視線上備份服務（獨有特徵）
- **中年群組**：平衡考慮娛樂和安全
- **青年群組**：較少關注備份，更重視設備保護
- 建議：對老年客戶強調資料安全和備份功能

#### 4. 技術支援的代溝
- **青年和中年**：願意嘗試技術支援計劃
- **老年**：較少使用技術支援
- 可能原因：老年用戶選擇較簡單服務，減少技術支援需求
- 建議：為老年用戶提供更易用的服務介面

#### 5. 多線路服務的普遍性
- **所有群組**都顯示多線路服務的高需求
- 反映多設備使用時代的特徵
- 建議：將多線路作為標準配置，並提供彈性升級選項

### 7.3 規則強度比較

**提升度排名：**
1. **中年群組**：平均 2.10，最大 3.73（關聯性最強）
2. **青年群組**：平均 1.98，最大 3.52
3. **老年群組**：平均 1.52，最大 2.33（關聯性最弱）

**解釋：**
- 中年群組的服務選擇最具規律性和一致性
- 老年群組的服務選擇較分散，個人偏好差異大
- 青年群組介於兩者之間

## 八、商業建議

### 8.1 針對青年群組（≤40歲）

#### 推薦策略：
1. **全方位娛樂包**
   - 組合：電影 + 電視 + 音樂 + 無限資料下載
   - 強調：串流自由、多設備支援

2. **數位優先方案**
   - 預設無紙化計費
   - 提供 APP 管理界面
   - 線上客服優先

3. **設備保護升級**
   - 強調：多設備保護
   - 包含：手機、平板、電腦全覆蓋

4. **彈性服務組合**
   - 允許高度客製化
   - 提供按需增減服務的彈性

### 8.2 針對中年群組（41-65歲）

#### 推薦策略：
1. **家庭完整方案**
   - 組合：多線路 + 全娛樂 + 雙重保護（設備+備份）
   - 強調：全家適用、一次到位

2. **工作與生活平衡包**
   - 包含：網路服務 + 線上備份（工作）
   - 附加：娛樂服務（生活）
   - 強調：效率與休閒兼顧

3. **技術支援增值**
   - 提供：優先技術支援
   - 強調：快速解決問題，不影響工作

4. **穩定性與可靠性**
   - 強調：服務穩定性
   - 提供：服務品質保證

### 8.3 針對老年群組（>65歲）

#### 推薦策略：
1. **簡化服務包**
   - 組合：電視 + 電影 + 電話服務
   - 特點：清晰、易懂、無複雜選項

2. **安全保障方案**
   - 核心：線上備份服務
   - 附加：設備保護計劃
   - 強調：資料安全、放心使用

3. **易用性優先**
   - 提供：大字體界面
   - 簡化：操作步驟
   - 強調：簡單好用

4. **人工服務支援**
   - 提供：電話客服優先
   - 強調：真人協助、耐心服務
   - 避免：過度依賴線上工具

### 8.4 跨群組通用策略

#### 1. 娛樂內容為王
- **所有群組**都重視娛樂服務
- 投資：高品質影音內容
- 策略：持續更新內容庫

#### 2. 網路服務為基礎
- **所有群組**都依賴網路服務
- 策略：確保網路品質和穩定性
- 投資：基礎設施升級

#### 3. 多線路成標配
- **所有群組**都需要多線路
- 策略：將多線路納入基礎方案
- 優化：提供彈性升級選項

#### 4. 服務綑綁銷售
- 利用關聯規則進行交叉銷售
- 例如：購買電影節目時推薦音樂節目
- 設計：智能推薦系統

## 九、結論

### 9.1 主要發現

1. **共同點多於差異點**
   - 三個年齡群組在基礎服務需求上高度一致
   - 娛樂服務是所有群組的共同興趣
   - 網路和電話服務是必備基礎

2. **關鍵差異點**
   - **服務複雜度**：青年和中年喜歡複雜組合，老年偏好簡單
   - **線上備份**：老年群組的獨特需求
   - **技術支援**：青年和中年更願意使用

3. **市場機會**
   - 中年群組規模最大，關聯性最強，應為重點服務對象
   - 老年群組規模雖小但需求明確，可提供專屬方案
   - 青年群組追求創新，可作為新服務測試市場

### 9.2 實施優先級

**高優先級（立即執行）：**
1. 針對中年群組推出「家庭全能包」
2. 強化娛樂內容品質（所有群組）
3. 簡化老年群組的服務介面

**中優先級（3-6個月）：**
1. 開發智能推薦系統（基於關聯規則）
2. 針對青年群組推出客製化工具
3. 加強線上備份服務的推廣（特別是老年群組）

**低優先級（持續優化）：**
1. 收集各群組的滿意度回饋
2. 持續優化服務組合
3. 監控各群組規則變化趨勢
